# 30-Day MERN Practice Plan

This 30-day plan is designed to help learners build a strong MERN stack foundation through daily practice, hands-on mini-projects, and steady full-stack progress.

## Plan Overview

- Duration: 30 days
- Focus: JavaScript, Node.js, Express, MongoDB, React, and MERN project integration
- Structure: 5 weeks / 6 days per week with one review day
- Outcome: a functioning MERN app with frontend, backend, database, and deployment readiness

## Week 1: JavaScript and Web Fundamentals

### Day 1: JavaScript Basics
- Learn variables, data types, and operators.
- Practice: write small scripts for math, strings, and booleans.
- Deliverable: JS file with examples of arrays, objects, and control flow.

### Day 2: Functions and Scope
- Learn functions, parameters, return values, and scope.
- Practice: build reusable functions and simple calculators.
- Deliverable: utility functions for a small form validation script.

### Day 3: DOM Manipulation
- Learn DOM querying, events, and element updates.
- Practice: create a dynamic counter or to-do list.
- Deliverable: a mini browser app that updates the page with JavaScript.

### Day 4: Asynchronous JavaScript
- Learn callbacks, promises, and `async`/`await`.
- Practice: fetch data from a public API and log results.
- Deliverable: a script that requests API data and handles errors.

### Day 5: JavaScript Modules and Tooling
- Learn module imports/exports and basic development tooling.
- Practice: split code into modules and use npm scripts.
- Deliverable: a small project with organized JavaScript files and `package.json`.

### Day 6: Review and Reinforcement
- Review key JavaScript concepts.
- Practice: complete a mini challenge combining DOM and API calls.
- Deliverable: a working interactive page and a short summary of learned concepts.

## Week 2: Node.js and Backend Basics

### Day 7: Node.js Introduction
- Learn Node.js runtime, npm, and package management.
- Practice: install packages and run Node scripts.
- Deliverable: a `hello world` Node script and `package.json`.

### Day 8: Express Server Setup
- Learn Express routing, middleware, and request handling.
- Practice: create a simple Express server with routes.
- Deliverable: server with at least 3 API endpoints.

### Day 9: REST API Fundamentals
- Learn REST principles and HTTP methods.
- Practice: build CRUD routes for a resource.
- Deliverable: API with `GET`, `POST`, `PUT`, and `DELETE` endpoints.

### Day 10: Middleware and Validation
- Learn request parsing, middleware, and validation.
- Practice: add JSON parsing and error handling.
- Deliverable: Express app with middleware for body parsing and validation.

### Day 11: Connecting Node and Database
- Learn how backend apps connect to databases.
- Practice: use a database client library to connect to MongoDB.
- Deliverable: Node app that connects successfully to MongoDB.

### Day 12: Review and Reinforcement
- Review Node and Express concepts.
- Practice: expand API with middleware and error handling.
- Deliverable: a polished Express API ready for frontend use.

## Week 3: MongoDB and Data Modeling

### Day 13: MongoDB Basics
- Learn MongoDB documents, collections, and commands.
- Practice: insert, query, and update documents.
- Deliverable: basic MongoDB CRUD operations in a shell or script.

### Day 14: Data Modeling
- Learn how to design collections and document structure.
- Practice: model a sample app dataset (users, posts, tasks).
- Deliverable: a written schema plan and example documents.

### Day 15: MongoDB CRUD with Node
- Learn MongoDB CRUD in Node.js.
- Practice: use Mongoose or native driver for create/read/update/delete.
- Deliverable: Node endpoints connected to MongoDB.

### Day 16: Indexing and Aggregation
- Learn MongoDB indexes and aggregation pipelines.
- Practice: create indexes and build a basic aggregation query.
- Deliverable: indexed collection and a summary report query.

### Day 17: Authentication Basics
- Learn authentication concepts for MERN apps.
- Practice: add user registration and login routes.
- Deliverable: backend login system with hashed passwords.

### Day 18: Review and Reinforcement
- Review MongoDB and backend integration.
- Practice: secure routes and database operations.
- Deliverable: a stable backend ready for frontend integration.

## Week 4: React and Frontend Development

### Day 19: React Fundamentals
- Learn components, JSX, and props.
- Practice: build reusable UI components.
- Deliverable: a React app with multiple components.

### Day 20: State and Effects
- Learn `useState` and `useEffect`.
- Practice: manage component state and fetch data.
- Deliverable: a React feature that loads data from an API.

### Day 21: Forms and User Input
- Learn controlled components and form handling.
- Practice: build a form and validate user input.
- Deliverable: a React form for creating or editing items.

### Day 22: React Routing
- Learn client-side routing with React Router.
- Practice: add multiple pages and navigation.
- Deliverable: a multi-page React app with routes.

### Day 23: Connecting to Your Backend
- Learn how to call your Express API from React.
- Practice: fetch and submit data from the frontend.
- Deliverable: React app connected to backend CRUD endpoints.

### Day 24: Review and Reinforcement
- Review React patterns and API integration.
- Practice: polish UI and fix frontend bugs.
- Deliverable: a responsive frontend ready for full-stack integration.

## Week 5: Full Stack MERN Integration and Deployment

### Day 25: MERN Project Setup
- Learn full-stack project structure.
- Practice: connect React, Express, and MongoDB in one project.
- Deliverable: working MERN app skeleton.

### Day 26: Implement Core Features
- Learn how to build end-to-end functionality.
- Practice: implement features like tasks, posts, or profiles.
- Deliverable: working feature with UI, API, and database.

### Day 27: Authentication and Authorization
- Learn JWT or session-based auth for MERN.
- Practice: secure routes and protect frontend views.
- Deliverable: authenticated MERN app flow.

### Day 28: Testing and Debugging
- Learn basic testing and debugging strategies.
- Practice: test API endpoints and frontend flows.
- Deliverable: list of fixes and small test cases.

### Day 29: Deployment Preparation
- Learn deployment basics for full-stack apps.
- Practice: prepare environment variables and production config.
- Deliverable: deployment-ready MERN app structure.

### Day 30: Launch and Review
- Learn deployment and project review.
- Practice: deploy to a hosting provider or document deployment steps.
- Deliverable: deployed MERN app or a detailed deployment plan.

## Optional Extension

- Add advanced features like file uploads, real-time updates, or testing automation.
- Build a second MERN project with a different domain to reinforce concepts.

## Tips for Success

- Keep daily practice consistent.
- Build while learning; every concept should be applied.
- Use version control and commit often.
- Review the MERN project goals each week.
- Ask questions and iterate on your app daily.
