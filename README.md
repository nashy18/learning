## How to Use This Repository

- Start at a category `README.md` to view the full curriculum structure.
- Each topic folder contains a `course.md` curriculum with objectives, concepts, exercises, interview prep, and real-world examples.
- Use the learning diagrams to understand the recommended study pathway and dependencies between topics.

This repository is maintained to organize learning materials by topic, subtopic, and practical curriculum for beginner learners.
